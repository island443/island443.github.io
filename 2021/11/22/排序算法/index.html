排序算法

发表于 2021-11-22 更新于 2023-07-17 分类于 algorithm

选择排序

选择排序就是从数组中第0n-1位置上的数中取一个最小的放在第0位，然后再从1n-1位置上的数中再取一个最小的放在第1位，依次到排到最后…

选择排序的时间复杂度是O(N^2)

冒泡排序

冒泡排序是把数组中第0个数和第一个数比较，若arr[0]>arr[1],则交换位置，在把arr[1]和arr[2]比较，若arr[1]>arr[2],则交换位置，这样能把最大的数换到数组末尾，再从头进行比较到倒数第二个数，以此类推…

冒泡排序的算法复杂度是O(N^2)

插入排序

插入排序是先排好数组中的第01的数，再拿第2个数和前面比较是否交换，这样就排好了02个数。再拿第3个数和前面进行比较，以此类推…

插入排序是不稳定的排序，理想的时间复杂度为O(N)，最坏情况下的时间复杂度为O(N^2)。

合并排序

合并排序的时间复杂度是O(N*log2N)

堆排序

堆排序是利用堆这种数据结构而设计的一种排序算法，堆排序的时间复杂度是O（N*logN），额外空间复杂度是O（1），是一个不稳定的排序。

1.堆的结构

堆的结构可分为大根堆和小根堆

每个结点的值都大于其左孩子和右孩子结点的值，称之为大根堆。每个结点的值都小于其左孩子和右孩子结点的值，称之为小根堆，如下图：

这里有个知识点：如果查找数组中某个数的父结点或者其左右孩子结点，比如索引为i的数，那么:

父结点索引：（i-1）/2

左孩子索引：2*i+1

右孩子索引：2*i+2

2.堆排序步骤

1.将待排序数构成大根堆（或小根堆）

2.将顶端和末尾数交换，此时末尾数为最大值，剩余待排数n-1个

3.将剩余n-1个数再构造大根堆，再将顶端数与n-1位置交换，如此反复，可得有序数组

这个过程称之为堆排序

由此，实现堆排序需要解决两个问题：（1）如何由一个无序序列建成一个堆？（2）如何在输出堆顶元素后,调整剩余元素成为一个新的堆？

先看第二个问题。

(a)是个堆，假设输出堆顶元素之后，以堆中最后一个元素替代之，如图（b)所示。此时根结点的左、右子树均为堆，则仅需自至下进行调整即可。首先以堆顶元素和其左、右子树根结点的值比较之，由于右子树根N点的值小于左子树根结点的值且小于根结点的值，则将27和97交换之；由于97替代27之后破坏了右子树的“堆”，则需进行和上述相同的调整，直至叶子结点，调整后的状态如图(c)所示，此时堆顶为n-1个元素中的最小值。重复上述过程，将堆顶元素27和堆中最后一个元素97交换且调整，得到如图(d)所示新的堆。

把这个从顶点至叶子的调整过程称为“筛选”

从一个无序序列建堆的过程就是一个反复筛选的过程，从第一个非终端节点n/2开始，依次按结点往上筛选
